Question-1: What are the different ways to select an element in the DOM?

Ans:- getElementByID, getElementByClassName, getElementByName, getElementByTagName, querySelector, querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:- innerHTML can parse HTML, it can show hidden text. innerText cannot parse HTML, it cannot show the hidden text. textContent cannot parse HTML, it can show the hidden text.

Question-3: What is event delegation in the DOM?

Ans:- we do event delegation in DOM to add event listener in a efficient way by targeting a parent element of many childs. we use event delegation in ul, table, div.

Question-4: What is event bubbling in the DOM?

Ans:- when we click into a nested element, the child will execute the first listener then the parent element will recive the event. we can stop it by using stopPropagation() methode.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans:- to create document.createElement(), to add append() or appendChild() , to remove removeChild() or remove().
